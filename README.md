# Student Management System

This project is a Java-based desktop application developed to digitize student data management in educational institutions. It features a user-friendly graphical user interface (GUI) focused on simplifying basic management processes.

Developed as part of an academic study (School Project), this application is designed to practice object-oriented programming (OOP) principles and fundamental Java libraries.

## 🚀 Features

* **Student Registration Management:** Adding new students, updating and deleting existing information (CRUD).
* **Fast Listing:** Dynamically displaying all registered students on the interface.
* **User-Friendly GUI:** Interface design that allows users to easily perform operations through windows and buttons without needing a command line.
* **Lightweight and Portable Architecture:** Directly executable structure without the need for heavy dependencies.

## 🛠 Technologies Used

* **Language:** Java (SE)
* **GUI (Interface):** Java Swing / AWT libraries
* **Distribution Format:** Executable JAR
